# Структура проекту SupportAI

Детальний опис файлів та директорій проекту.

## Коренева директорія

| Файл / Директорія | Опис |
| :--- | :--- |
| `generate.py` | Точка входу для генерації датасету. |
| `analyze.py` | Точка входу для аналізу датасету. |
| `pyproject.toml` | Конфігурація проекту, залежності та налаштування інструментів (Ruff, MyPy). |
| `requirements.txt` | Список залежностей (експортовано з `pyproject.toml`). |
| `Dockerfile` | Інструкції для створення Docker-образу додатку. |
| `docker-compose.yml` | Конфігурація сервісів (app, ollama, ollama-pull-model). |
| `.pre-commit-config.yaml` | Налаштування git-хуків для перевірки коду перед комітом. |
| `README.md` | Головна документація проекту (англійською). |
| `docs/` | Директорія з детальною документацією процесів (українською). |
| `data/` | Директорія для збереження згенерованих даних та результатів аналізу. |
| `logs/` | Директорія для логів роботи додатку. |
| `src/` | Вихідний код проекту. |

## Вихідний код (`src/`)

### `src/generator/` (Логіка генерації)

- `main.py` — Оркестратор процесу генерації: завантаження бази, аналіз вибірки, розширення та збереження.
- `engine.py` — Ядро генерації: робота з Bitext, LLM-запити для варіацій та створення складних сценаріїв.
- `prompts.py` — Шаблони промптів для LLM, що використовуються при генерації.
- `__init__.py` — Маркер пакету.

### `src/analyzer/` (Логіка аналізу)

- `main.py` — Оркестратор процесу аналізу: завантаження `dataset.json`, батчевий аналіз через LLM та розрахунок статистики.
- `engine.py` — Ядро аналізу: валідація відповідей LLM, нормалізація метрик та розрахунок фінальних оцінок.
- `prompts.py` — Шаблони промптів для LLM, що використовуються при аналізі діалогів.
- `__init__.py` — Маркер пакету.

### `src/config/` (Конфігурація)

- `constants.py` — Глобальні константи: мапінг інтентів, типи помилок агентів, SEED для детермінізму.
- `logger.py` — Налаштування логування через `loguru` (вивід у консоль та файли).
- `__init__.py` — Маркер пакету.

## Дані (`data/`)

- `dataset.json` — Фінальний згенерований датасет діалогів.
- `dataset_stats.json` — Статистика за результатами генерації.
- `analysis.json` — Результати незалежного аналізу діалогів.
- `analysis_stats.json` — Агрегована статистика аналізу та порівняння з генератором.
